<h3>On Windows:</h3>

* %USERPROFILE%\\_vimrc
* %USERPROFILE%\\.vim\bundle\vundle should be installed (https://github.com/gmarik/vundle/wiki/Vundle-for-Windows)
* :BundleList to confirm bundles in _vimrc can be found
* :BundleInstall and you should be sorted
